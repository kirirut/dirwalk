О чём программа?

Программа представляет собой утилиту для рекурсивного обхода директорий. Она анализирует содержимое указанной директории и выводит информацию о файлах, каталогах и символических ссылках в зависимости от переданных параметров.
Зачем нужна программа?
Программа полезна для анализа структуры директорий и получения информации о файлах, каталогах и ссылках. Она может быть использована для:

Поиска файлов определённого типа (например, только каталогов или только символических ссылок).

Рекурсивного обхода вложенных директорий.

Вывода полных путей к объектам.

Как работает программа?
Аргументы командной строки:
Программа принимает следующие параметры:

-l: Показывать символические ссылки.
-d: Показывать каталоги.
-f: Показывать файлы.
-s: Сортировать вывод (в текущей реализации сортировка не реализована).
<dir>: Путь к директории, которую нужно обойти.

Если ни один из флагов (-l, -d, -f) не указан, программа по умолчанию показывает все типы объектов (файлы, каталоги и ссылки).

Основные функции:

main:

Обрабатывает аргументы командной строки.

Проверяет, указана ли директория для обхода.

Вызывает функцию walk_directory для начала обхода.

walk_directory:

Открывает указанную директорию с помощью opendir.

Рекурсивно обходит все элементы директории с помощью readdir.

Игнорирует текущую (.) и родительскую (..) директории.

Для каждого элемента вызывает функцию process_entry.

process_entry:

Формирует полный путь к элементу.

Использует lstat для получения информации о типе элемента (файл, каталог, символическая ссылка).

В зависимости от переданных флагов выводит информацию о элементе.

Если элемент является каталогом, рекурсивно вызывает walk_directory для его обхода.

Рекурсивный обход:
Программа рекурсивно обходит все вложенные директории, начиная с указанной, и выводит информацию о файлах, каталогах и ссылках.
